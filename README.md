## 🖥️ Overview
**JAT** is a lightweight, single-page framework designed to act as a centralized command center for web-based security tools. Instead of digging through bookmarks, JAT organizes high-value resources into a searchable, categorized interface.

### 🔍 Key Features
- **Instant Search:** Real-time filtering of the entire tool library.
- **Categorized Tree View:** Neatly organized folders for OSINT, CTFs, Exploits, and more.
- **Dynamic Counters:** Automatically tracks the number of tools in each category.
- **Zero Dependencies:** Pure HTML/CSS/JS. No database or server-side hosting required.

---

## 🛠️ Tech Stack
- **Frontend:** HTML5, CSS3 (Custom Dark Variable System)
- **Logic:** Vanilla JavaScript (ES6+)
- **Styling:** GitHub-inspired dark aesthetic with neon orange accents.

## 📂 Library Categories
| Category | Description |
| :--- | :--- |
| **OSINT** | Open Source Intelligence & Investigation tools (PimEyes, Shodan, etc.) |
| **STUDY** | Learning platforms and cheat sheets (TryHackMe, PortSwigger) |
| **CTF/BUG BOUNTY** | Platforms for active hacking practice and opportunities |
| **TOOLS** | Practical utilities (CyberChef, RevShells, GTFOBins) |
| **EXPLOIT** | Vulnerability databases and payload generators |
| **RECON** | Subdomain discovery and automation tools |

## 🚀 Getting Started (Locally)
1. **Clone the repository:**
  ``` git clone https://github.com/alreadyClosed/JAT.git```

2. **Launch:**
   Simply open index.html in any modern web browser.

3. **Usage:**
   Type any tool name (e.g., "Shodan" or "Hash") into the search bar to filter results instantly.

---

## 📝 Customization
To add your own links, simply edit the myTools array in the script tag of the index.html file:
```
const myTools = [
    ["CATEGORY", "Tool Name", "https://url-link.com"],
];
```
